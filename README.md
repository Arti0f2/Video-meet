# Video-Meet (Відеозустрічі)

Це сучасний веб-додаток для проведення відеоконференцій у реальному часі, створений за допомогою **React**, **Node.js**, **Express** та **Socket.io**. Додаток дозволяє користувачам створювати кімнати, ділитися посиланнями для приєднання, спілкуватися в чаті та обмінюватися відео/аудіо потоками.

## 🚀 Основні можливості

- **Відео та аудіо зв'язок:** Використання WebRTC для передачі медіаданих.
- **Текстовий чат:** Обмін повідомленнями та файлами в реальному часі.
- **Планування зустрічей:** Можливість створювати та переглядати заплановані конференції.
- **Запис зустрічей:** Збереження відеозаписів на сервері.
- **Доступ по локальній мережі:** Налаштований HTTPS дозволяє підключатися з різних пристроїв (телефони, планшети) в одній Wi-Fi мережі з повним доступом до камери та мікрофона.

## 🏗 Архітектура (OOP & SOLID)

Проект розбитий на дві незалежні частини:

1.  **Frontend (`/frontend`):** Побудований на React. Використовує компонентний підхід. Логіка розподілена по сторінках (`pages`) та компонентах.
2.  **Backend (`/backend`):** Побудований на Node.js. Використовує об'єктно-орієнтований підхід:
    - **Services:** Окрема логіка для роботи з даними (Meetings, Recordings).
    - **Controllers:** Обробка HTTP-запитів.
    - **SocketHandler:** Інкапсульована логіка веб-сокетів для сигналінгу та чату.

## 🛠 Технологічний стек

- **Frontend:** React, Material-UI, Socket.io-client, Bootstrap.
- **Backend:** Node.js, Express, Socket.io, Multer (для завантаження файлів).
- **Security:** Самопідписані SSL-сертифікати для роботи через HTTPS в локальній мережі.

## 🚦 Як запустити проект

### 1. Попередня підготовка
Переконайтеся, що у вас встановлено [Node.js](https://nodejs.org/).

### 2. Встановлення залежностей
З кореневої папки проекту виконайте команду, яка встановить пакети для фронтенду та бекенду, а також автоматично згенерує SSL-сертифікати:
```bash
npm run install-all
```

Якщо вам потрібно згенерувати сертифікати окремо, виконайте:
```bash
npm run cert
```

### 3. Запуск у режимі розробки
Для запуску проекту виконайте:
```bash
npm run dev
```

Після запуску в консолі з'являться посилання для доступу:
- **Локально:** `https://localhost:8000`
- **У мережі:** `https://<ваш-ip>:8000` (використовуйте це посилання для входу з інших пристроїв).

## ⚠️ Важливо щодо HTTPS

Оскільки додаток використовує самопідписані сертифікати (для забезпечення доступу до камери/мікрофона без справжнього домену):

1.  Браузер покаже попередження **"Your connection is not private"**. Це нормально. Натисніть **Advanced (Додатково)** -> **Proceed (Перейти)**.
2.  **ОБОВ'ЯЗКОВО:** Для роботи відео та чату потрібно також один раз зайти за адресою бекенду: `https://<ваш-ip>:4001` та також підтвердити виключення безпеки (натиснути Proceed).

## 📁 Структура папок

- `backend/src/services` — бізнес-логіка додатку.
- `backend/src/controllers` — обробка API маршрутів.
- `backend/src/sockets` — управління з'єднаннями Socket.io.
- `frontend/src/pages` — основні екрани (Головна, Кімната зустрічі).
- `backend/recordings` — папка, де зберігаються записані відео.
